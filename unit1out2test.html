Unit 2 - 1.2 Plan and create your website:

To plan my website I firstly looked at other websites to get ideas and researched HTML code to start coding the barebones of the web pages. I started off implementing the HTML code into Visual Studio Code, that then created a basic web page with a white background and black standard text with no text alignment. I then started researching CSS and created a CSS file. I linked the CSS file to my HTML file and started inputting code into CSS to design the web page. I added a background image and sorted out the text alignment to how I wanted it. I used the CSS file in conjunction with the HTML files I created for each page to add colour and structure to my website. I pushed all my website content periodically to my GitHub account where it is stored.


Unit 2 - 1.3 Selecting and using web site features and structure:

I decided to create a fixed navbar which would be situated to the top of each web page within my website. On the navbar I placed a logo I created to the left and three links which navigate to the three main pages within my website structure. I copied the HTML code from the index.html page and pasted it into every page on my website so the navbar would consistently show up. I edited the HTML code in each page to take into accountwhat pages they are, the content they contain and how the pages navigate to each other.


Unit 2 - 1.4 Selecting and using styles to keep your site consistent:
The style I went for was the same blue/black background image for each web page within my website structure. I also created a navbar with its own green/black background image, which also contained a logo I created and three main links that navigated to other web pages within my website structure. I set all ‘H1’, ‘H2’ and ‘H3’ tags in my CSS file to show a different font type, font size and colour (I also added a shadow to all text contained within my website. I added a hover feature on the navbar, so when you hover over a link within the navbar it highlights it with a transparent colour.


Unit 2 - 1.8 Storing and retrieving files effectively:
Whilst creating my website using Visual Studio Code I setup a folder on the desktop of the computer I was using. This stored all files needed for my website. I also linked my GitHub account to Visual Studio Code and pushed all content to my GitHub repository. I do a push to my repository after I make a change in any code on my website so its always up to date.
