1.5 Describing copyright and other constraints affecting websites

Copyright and other constraints like intellectual property laws affect websites, as they can restrict what you are legally allowed to publish. Copyright is a means of protecting original works by individuals or organisations and can only be used by outside sources with expressed permission from the original author(s). Copyright covers both published and unpublished works.

1.6 Describing access issues that need taking into account - describe issues that people with disabilities may face when using web pages and how we can overcome them

Disabilities of different kinds affect people in many different ways when trying to access websites, for example; a blind person would not be able to see the display and a way of overcoming this issue would be to rely on either on-screen readers and/or refreshable braille displays. Another example; an individual with physical disabilities might not be able to use the hardware to navigate the device they are using and so a way of overcoming this issue would be to rely on voice-recognition software to take commands from the user by talking to the device.


Reference for this answer;
(https://www.w3.org/WAI/EO/Drafts/WAI-access-profiles)




1.7 Describe which are the appropriate filetypes for websites

There are many different file types which can be appropriately used for websites. HTML (Hypertext Markup Language) is both a file format and a language in which a web page is written in. There are several image formats which can be used for websites, common file types include JPEG (Joint Photographic Experts Group) and PNG (Portable Network Graphics). The difference between these two file types is JPEG is used for images containing a lot of different colours, for example; photographs. It is also a good image file type to save storage space, as it can reduce the size of highly detailed image files. PNG is used for images that contain transparent backgrounds or small images, for example; logos. Like image file formats there are also several choices when it comes to audio file formats, common file types include WAV and AAC (Advanced Audio Coding). The difference between these two file types is WAV is uncompressed audio (which is virtually lossless), is used for short sound effects in applications and raw sound samples. AAC is like MP3 but was designed to be an improvement over that file format. It is lossy compressed but, can produce better sound quality at the same file size and bitrate as compared to MP3. There are also several video file formats which can be used, common file types include WebM and AVI (Audio Video Interleave). WebM is quite a new open video file format and is used for embedded video clips which are short in length to work in conjunction with HTML5 technologies. AVI is quite an old video file format which is uncompressed and is widely popular in file sharing communities. There are many more file format types than I can mention for using on a website, but the ones I have mentioned are only a small amount. To find out all file types you can search online for the extensive list.
